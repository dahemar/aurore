Google Sheets CMS Guide for Aurore

How to edit website content using a public Google Sheet.

Quick links

  • Spreadsheet: add your link (e.g., https://docs.google.com/...)
  • Tabs: page1_je_mappelle_aurore, page2_topographie_etrange, page3_reliques_reve, page4_memoires_mont_songe

1) Visibility

  • Share → General access → Anyone with the link (Viewer)
  • No backend required; the app reads CSV exports of each tab

2) Tab schemas

page2_topographie_etrange

  • image_url – public image URL
  • caption – short text
  • size – normal or big

Positions are randomized; coordinates are not read from the sheet.

page4_memoires_mont_songe

  • text_type – introductory_quote | main_content
  • content – text; line breaks are preserved

3) Images

  • Host in react-app/public/images and reference via ${BASE_URL}images/<file>, or use an external host
  • Recommended width: 800–1600px (JPG/WEBP)

4) How it works

https://docs.google.com/spreadsheets/d/<SHEET_ID>/gviz/tq?tqx=out:csv&sheet=<TAB_NAME>

Frontend hook: useSheetData('<tab_name>'). Refresh the page to see edits immediately.

5) Editing steps

  1. Open the sheet → select tab
  2. Edit only the required columns
  3. Keep the first row as headers
  4. Reload the site page

6) Troubleshooting

  • Empty page → check tab names exactly
  • No image → the URL must be public and direct
  • No line breaks in the quote → use real line breaks in the cell

7) Export to PDF

Use your browser’s Print → Save as PDF. This page is formatted for A4 printing.
